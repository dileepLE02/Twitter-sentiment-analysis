# Twitter Sentiment Analysis using NLTK

This project performs sentiment analysis on Twitter data using Natural Language Toolkit (NLTK), TF-IDF Vectorization, and Machine Learning models including Logistic Regression, Bernoulli Naive Bayes, and Support Vector Machine (SVM).

## Features
- Exploratory Data Analysis (EDA)
- Text preprocessing using NLTK
- Tokenization
- Stopword Removal
- Stemming and Lemmatization
- TF-IDF Feature Engineering
- WordCloud Visualization
- Model Evaluation using Accuracy, Confusion Matrix, and ROC-AUC Curve

## Technologies Used
- Python
- NLTK
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Models Used
- Bernoulli Naive Bayes
- Logistic Regression
- Support Vector Machine (SVM)

## Conclusion
The project successfully classifies Twitter sentiments into positive and negative categories using NLP preprocessing and machine learning techniques. Logistic Regression demonstrated strong overall performance for sentiment classification tasks.
